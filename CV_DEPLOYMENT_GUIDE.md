# CV Generator App Deployment Guide

## Deploy to Vercel (5 minutes)

### Prerequisites
- GitHub account (you have this ✓)
- Vercel account (free: https://vercel.com/signup)

### Step 1: Connect to Vercel
1. Go to https://vercel.com
2. Click **Sign Up** → Sign up with GitHub
3. Authorize Vercel to access your GitHub repos

### Step 2: Import Project
1. Click **Add New** → **Project**
2. Search for `cv-generator-app`
3. Click **Import**
4. Leave settings as default
5. Click **Deploy**

**That's it!** Vercel will automatically build and deploy your app in ~1-2 minutes.

### Step 3: Get Your Live URL
After deployment completes:
- Vercel gives you a URL like: `https://cv-generator-app-xxxxx.vercel.app`
- Copy this URL

### Step 4: Update README
Add to `cv-generator-app/README.md`:

```markdown
## Live Demo

[View Live Demo](https://your-deployment-url.vercel.app)
```
